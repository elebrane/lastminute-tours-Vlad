# REFERENCES.md — Обзор литературы и похожих решений на рынке

## 1. Обзор существующих решений

### 1.1 Прямые конкуренты (горящие туры)

#### Level.Travel (level.travel)
**Описание:** Крупнейший российский онлайн-агрегатор туров. Поиск по 60+ туроператорам, есть раздел горящих туров.  
**Сильные стороны:**
- Огромная база туров
- Фильтры по направлениям, отелям, датам
- Мобильное приложение

**Слабые стороны:**
- Перегружен информацией
- Нет персонализации для спонтанных путешественников
- Нет AI-описаний для быстрого принятия решения

#### Travelata (travelata.ru)
**Описание:** Поисковик туров с фильтром горящих предложений.  
**Сильные стороны:** Понятный интерфейс, быстрый поиск  
**Слабые стороны:** Нет ИИ-рекомендаций, нет фильтра "на эти выходные"

#### Last Minute Club (lastminuteclub.ru)
**Описание:** Агрегатор горящих туров с email-рассылками.  
**Сильные стороны:** Фокус на горящих турах  
**Слабые стороны:** Устаревший UX, нет мобильной версии, нет AI

#### Onlinetours (onlinetours.ru)
**Описание:** Мета-поисковик туров, включает горящие предложения.  
**Сильные стороны:** Большой выбор, сравнение цен  
**Слабые стороны:** Нет специализации на "последней минуте", нет AI

### 1.2 Международные аналоги

#### Hotwire (hotwire.com)
**Описание:** Американский сервис горящих предложений (отели, авиа, пакеты).  
**Ключевая особенность:** "Тайные цены" — видишь отель только после покупки.

#### LastMinute.com
**Описание:** Крупнейший европейский сервис горящих предложений.  
**Ключевая особенность:** Специализация именно на последней минуте, раздел "Weekend"

#### Secret Escapes
**Описание:** Клубный сервис с закрытыми скидками для зарегистрированных пользователей.  
**Ключевая особенность:** Кураторские подборки, высокое качество отелей

### 1.3 AI-помощники для путешествий

#### Wonderplan.ai
AI генерирует маршруты путешествий на основе предпочтений. Не занимается поиском туров, только планированием.

#### Roam Around
AI-планировщик путешествий (ChatGPT-based). Создает дневные маршруты по городу.

#### GuideGeek (TripAdvisor)
Туристический AI-ассистент в мессенджерах. Отвечает на вопросы о путешествиях.

## 3. Научные работы и статьи

1. **"Last Minute Tourism: Understanding the Decision-Making Process"** (Journal of Travel Research, 2018) — исследование показывает, что 34% туристов принимают решение о поездке менее чем за 2 недели.

2. **"AI in Travel: Personalization and Recommendation Systems"** (Springer, 2022) — обзор применения машинного обучения для персонализации туристических рекомендаций.

3. **"The Psychology of Spontaneous Travel"** (Annals of Tourism Research, 2020) — факторы, влияющие на импульсивные туристические решения.

4. **"LLMs for Tourism Recommendation"** (arXiv, 2023) — использование больших языковых моделей для генерации туристических описаний и рекомендаций.

5. **"Price Sensitivity in Last-Minute Hotel Bookings"** (Cornell Hospitality Quarterly, 2019) — анализ ценовой чувствительности при бронировании горящих предложений.

## 4. API и технологии

### Level.Travel API
- **URL:** https://api.level.travel/
- **Документация:** Партнерский доступ
- **Возможности:** Поиск туров, цены, доступность, бронирование
- **Формат:** REST API, JSON

### OpenAI API (через Replit AI Integrations)
- **Модель:** gpt-5-mini (оптимально для генерации описаний)
- **Применение:** Генерация кратких описаний туров и вердиктов
- **Задержка:** ~1-2 секунды на описание
